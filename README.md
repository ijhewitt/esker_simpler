esker_simpler.m is the main code, which is called by a run file.  
esker_simpler_run_dim.m is the main example run file.  Inputs and outputs are in physical units.
esker_simpler_run.m is an alternative example run file, which performs non-dimensionalisation outside the main code and then works with non-dimensional inputs/outputs.
esker_simpler_py contains similar files ported to python with the help of Claude
